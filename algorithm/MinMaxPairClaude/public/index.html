How It Works

This tool finds pairs of elements in your array where:

  • The minimum difference between array[i] - array[j] is found (where j > i)
  • The maximum difference between array[i] - array[j] is found (where j > i)
  • If multiple pairs exist with the same difference, the pair with the smallest indices is chosen

All indices are reported using 1-based indexing for readability.
