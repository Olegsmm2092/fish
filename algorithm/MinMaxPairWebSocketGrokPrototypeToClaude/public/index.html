Problem Description

Given an array of integers, find two indices i and j (where j > i) such that:

  • The difference elements[i] - elements[j] is minimized.
  • The difference elements[i] - elements[j] is maximized.
  • If multiple pairs exist, choose the pair with the smallest i, then smallest j.
  • Output indices as 1-based.

Example 1

Input: 6\n2 1 3 5 2 4\nOutput: 2 4\n4 5

Example 2

Input: 5\n3 2 4 5 6\nOutput: 2 5\n1 2
